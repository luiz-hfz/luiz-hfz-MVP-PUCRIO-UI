# MVP - Protótipo de Aplicativo para Agendamento de Consultas

## Autor

Ana Cristina Fernandes Lima

## Instituição

Pontifícia Universidade Católica do Rio de Janeiro (PUC-Rio)

## Curso

Pós-Graduação em Interação Humano-Computador e Experiência do Usuário (UX)

## Contexto do Trabalho

O presente repositório tem como objetivo apresentar o desenvolvimento de um MVP de interface para um aplicativo de agendamento de consultas em clínicas, com foco na experiência do usuário durante o fluxo de busca, seleção e confirmação de uma consulta.

O projeto contempla a modelagem da interação por meio da MoLIC, a definição de cenários de uso, a criação de um mini Design System e o desenvolvimento de um protótipo de alta fidelidade no Figma.
